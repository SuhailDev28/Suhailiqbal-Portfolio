# Suhail Iqbal — Animated Full Stack Developer Portfolio

Premium animated React/Vite portfolio with Framer Motion.

## New Features

- Framer Motion scroll reveal animations
- Animated hero card and floating tech icons
- Project filtering by category
- Theme toggle with localStorage
- Animated scroll progress bar
- Floating quick-action dock
- Animated stats counters
- Expanded services section
- Grouped tech stack cards
- Process cards with descriptions
- Testimonials/client-value section
- Responsive mobile menu
- Dark/light theme support

## Install

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Upload the generated `dist` folder to Netlify, Vercel, Hostinger, or Render Static Site.


## 3D Upgrade

This version includes lightweight CSS 3D elements:
